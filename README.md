# Interactive-App-Data-Analysis-and-Visualization-System-Using-Python
In this project, I developed a menu-driven system that stores app-related data such as ratings, reviews, installs, size, and price. The system allows users to edit the data, perform statistical analysis, and visualize the data using different types of graphs like bar charts, scatter plots, pie charts, histograms, and stem plots.
